# -SQL-Project
This is a data cleaning project carried out using SQL. It highlights practical methods to clean, structure, and prepare raw data for meaningful insights.
The project applies four key data cleaning steps:

Removing duplicates – Ensures each record is unique.

Standardizing data – Formats text, dates, and other fields consistently.

Handling null and blank values – Fills or removes missing data.

Removing unnecessary rows or columns – Keeps the dataset relevant and manageable.

Technologies Used:

SQL (Structured Query Language)

MySQL (Database management system)
